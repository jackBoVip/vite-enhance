# Roadmap Documentation

## 发展路线图

Vite Enhance Kit 的发展分为三个主要阶段，逐步构建完整的前端工程化解决方案。

## Phase 1: 核心能力 (Core Capabilities)

**目标**: 建立稳定的核心架构和基础功能

### 核心功能
- [x] Monorepo 基础架构
- [ ] 智能项目类型检测 (app/lib/monorepo)
- [ ] 框架自动识别 (Vue/React/Vanilla)
- [ ] 插件生命周期系统
- [ ] 配置 API 和校验

### 预设系统
- [ ] App 预设 (Web 应用构建)
- [ ] Lib 预设 (库/SDK 构建)
- [ ] 预设配置覆盖机制

### 官方插件
- [ ] Framework Vue (Vue 3 SFC 支持)
- [ ] Framework React (Fast Refresh 支持)
- [ ] CDN 插件 (公网 + 内网支持)
- [ ] 构建缓存插件
- [ ] 包分析插件

### CLI 工具
- [ ] `vek dev` - 开发服务器
- [ ] `vek build` - 生产构建
- [ ] `vek preview` - 预览服务器
- [ ] `vek analyze` - 包分析

**里程碑**: 支持单仓库 Vue/React 应用的完整开发流程

## 版本发布策略

### 核心包版本同步
- `@vite-enhance/core`
- `@vite-enhance/cli`
- `@vite-enhance/config`

### 插件独立迭代
- 官方插件可独立发布
- 向后兼容性保证
- 语义化版本管理

### 发布周期
- **Major**: 6-12 个月 (破坏性变更)
- **Minor**: 1-2 个月 (新功能)
- **Patch**: 按需发布 (Bug 修复)
